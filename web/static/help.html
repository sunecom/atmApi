OpenClaw Gateway 配置

编辑 OpenClaw 配置文件（openclaw.json），添加 atmapi provider：

{ "providers": { "atmapi": { "baseUrl": "https://atmapi.aitomoney.online/v1", "apiKey": "***" } }, "agents": { "defaults": { "model": { "primary": "atmapi/deepseek-a4" } } } }
💡 配置后重启 Gateway：openclaw gateway restart，之后所有对话自动走 deepseek-a4 智能路由。

OpenClaw MCP 集成

也可以只通过 MCP 接入 atmApi 工具（查询用量、续费等），模型仍用自己的：

{ "mcpServers": { "atmapi": { "type": "http", "url": "https://atmapi.aitomoney.online/mcp", "headers": { "Authorization": "***" } } } }

Hermes（小马哥）配置

Hermes 用户使用相同的 API 地址和 Token，模型名由 Hermes 工作流自动指定，无需手动配置。

配置文件位置：~/.hermes/config.yaml

# ~/.hermes/config.yaml api: base_url: https://atmapi.aitomoney.online/v1 api_key: "***" model: deepseek-a4
💡 注意：Hermes 套餐（基础 ¥149/进阶 ¥399）和 DeepSeek-A4 套餐不通用。请确认您的 Token 属于 Hermes 产品线。如有问题，在群内联系小马哥。

❓ 常见问题

Q: deepseek-a4 是什么模型？

A: 一个智能路由层，自动选择 flash/pro/qwen，您不需要手动切换。

Q: 配额用完了怎么办？

A: 滑动窗口自动重置。5小时窗口滚动即可恢复，月卡有效期内持续有效。

Q: Token 会过期吗？

A: 激活后 30 天有效。到期前 7 天自动提醒续费。

Q: 如何续费或升级？

A: 对话中发送「续费」或「升级」，或访问 Token 查询页面。

Q: 不同产品线的套餐能混用吗？

A: 不能。DeepSeek-A4、OpenClaw（atm卡）、Hermes 是独立产品线，套餐互不通用。

⚠️ 注意事项

  • GLM-5.2 套餐与 DeepSeek-A4 套餐不通用，请确认购买正确的产品线
  • GLM-5.2 是推理模型，首次响应可能较慢（需要 reasoning），属正常现象
  • GLM-5.2 是纯文本模型，不支持图片识别。如需图片理解，请使用 DeepSeek-A4
  • 点数按滑动窗口自动重置，未用完不累积
  • Token 是您的专属密钥，请勿泄露
  • 禁止用于违法、违规内容

常见错误码

错误码含义处理方式
400请求格式错误检查 messages 格式是否符合 OpenAI 规范
401Token 无效或未激活检查 API Key 是否正确，首次使用会自动激活
429触发限流5h/日/月额度用完，等待窗口恢复
402点数不足充值或升级套餐
502上游渠道故障系统会自动切换渠道，如持续报错请联系客服
400 + 不支持图片发送了图片GLM-5.2 不支持图片，请改用 DeepSeek-A4

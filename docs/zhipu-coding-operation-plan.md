## 🛠️ 3. atmApi 配置清单

### 3.1 渠道配置 (Channels)
需要在 atmApi 后台新增一个专用渠道：
*   **名称**: 智谱 Coding (Team)
*   **Base URL**: `https://open.bigmodel.cn/api/paas/v4` (需确认最终端点)
*   **模型映射**: 
    *   `glm-5.2` -> `glm-5.2` (或官方指定的 Coding 模型名)
*   **优先级**: 设为最高 (Priority: 10)，作为首选通道。

### 3.2 Token 模板 (MAS Tokens)
在数据库 `mas_tokens` 表中预设三种模板：
1.  **Starter**: `quota=500`, `reset_interval=5h`
2.  **Pro**: `quota=1000`, `reset_interval=5h`
3.  **Max**: `quota=1500`, `reset_interval=5h`

### 3.3 自动化任务 (Cron)
*   **配额重置**: 每 5 小时扫描一次活跃 Token，恢复其 `remain_quota`。
*   **用量监控**: 每小时统计一次智谱渠道的总消耗，防止超支。

## ⚠️ 4. 风险控制与应对

### 4.1 并发限制 (QPS)
*   **风险**: 2 个席位通常只支持 2-5 个并发会话。
